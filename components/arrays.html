  • Two Number sum.

    Create a function that takes in an array of distinct integers and an integer representing a target sum. The function should return an array containing any two numbers from the input array that sum up to the target sum. If no two numbers sum up to the target sum, the function should return an empty array.

    Make a list of numbers. Then, pick two numbers from the list that add up to a certain number. If you can’t find two numbers that add up to that number, then you don’t have any numbers on your list.
